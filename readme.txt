Plugin dodający widget callweb do strony internetowej. Dzięki widgetowi możesz automatycznie oddzwaniać do swoich klientów.

== Description ==

= Generuj leady! =

Dzięki słuchawce umieszczonej na Twojej stronie każdy użytkownik może zrealizować szybkie połączenie z Twoją firmą – kontakt następuje do 22 sekund! Co więcej, każdy może pozostawić leada w postaci numeru telefonu w celu późniejszego oddzwonienia.

= Jak to działa? To proste! =

* Widget będzie widoczny na Twojej stronie internetowej.
* Klient wpisze swój numer telefonu.
* Aplikacja automatycznie połączy Klienta z Twoim pracownikiem.

= Większa sprzedaż =

Widoczna słuchawka na Twojej stronie wygeneruje jeszcze większą liczbę leadów od potencjalnych klientów

= Oszczędność czasu =

W biznesie każda minuta ma znaczenie, dlatego pozwól swoim klientom na szybki kontakt z Twoją firmą

= Efektywność pracy =

Miej pod kontrolą jakość pracy swoich Konsultantów, co wpłynie na efektywność ich pracy

= Komfort pracy =

Twoi klienci nie będą musieli czekać w kolejkach, wystarczy, że zamówią połączenie w wygodnym dla siebie terminie

[Zarejestruj konto za darmo](https://app.callweb.pl/ "Aplikacja Callweb")

== Installation ==

= Za pomocą wyszukiwarki =

Najłatwiejszym sposobem na zainstalowanie wtyczki jest użycie zaplecza administracyjnego Wordpress.

* Przejdź do panelu administracyjnego Wordpress
* Znadź w menu zakładkę wtyczki/plugins
* Kliknij "dodaj nową"
* Wyszukaj Callweb
* Kliknij zaintsaluj
* Przejdź do listy wtyczek
* Kliknij "Aktywuj" pod nazwą Callweb
* Zaloguj się na swoje konto callweb i skopiuj klucz widgetu
* Wklej klucz widgetu w ustawieniach Wordpress. Znajdziesz je w menu pod nazwą Callweb. Nie zapomnij zaznaczyć checkbox'a "Czy dodać kod widgetu na stronę?"

== Frequently Asked Questions ==

= Jak zainstalować aplikację call web? =

1. Dodaj adres strony internetowej w zakładce “widget”, na której ma być widoczny widget (w aplikacji).
2. Zaintaluj wtyczkę Callweb na swojej stronie Wordpress.
3. Skopiuj klucz widgetu z aplikacji i wklej go w ustawieniach wtyczki.
4. Dodaj i zweryfikuj numer telefonu. (w aplikacji)
5. Dodaj konsultanta. (w aplikacji)
6. Dostosuj widget do swojej strony internetowej. (w aplikacji)
